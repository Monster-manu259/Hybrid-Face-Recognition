How It Works

1
Reference photo is loaded & face is detected with RetinaFace
2
FaceNet encodes it into a 512-dimensional L2-normalized vector
3
Pinecone ANN search retrieves top-K nearest stored embeddings
4
Matches below distance threshold are temporally clustered into appearance segments

Threshold Guide

0.30–0.40 — Strong match
0.40–0.50 — Good match (default)
0.50–0.60 — Loose match
